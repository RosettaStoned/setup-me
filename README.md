# setup-me
Setup oh-my-zsh and space-vim.

### Basic Installation

#### via curl

```shell
sh -c "$(curl -fsSL https://raw.githubusercontent.com/RosettaStoned/setup-me/master/setup.sh)"
```

#### via wget

```shell
sh -c "$(wget https://raw.githubusercontent.com/RosettaStoned/setup-me/master/setup.sh -O -)"
```
